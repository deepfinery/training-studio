sidebar_label: Navigation
---

- **Top bar**: workspace switcher (if multi-tenant), user menu (profile, logout).  
- **Sidebar**: primary app areas in order of use:
  - **Data**: upload and manage source documents.
  - **Training**: configure and launch fine-tuning jobs.
  - **Evaluation**: test outputs against prompts/cases.
  - **Deploy**: download artifacts or push to serving targets.
  - **History**: audit of uploads, jobs, evaluations, and deployments.
  - **Settings**: org/user preferences, API base, auth settings.
- **Notifications**: job progress and completion toasts for long-running tasks.
